# Static Website Starter

A lightweight starter template for a responsive static website.

## What is included

- `index.html` — homepage markup with clear sections
- `styles.css` — mobile-first layout and modern visuals
- `script.js` — small script for mobile navigation and current year update

## Getting started

1. Open `index.html` in your browser.
2. Edit the page content directly in `index.html`.
3. Customize colors and spacing in `styles.css`.

## Notes

- This is a plain HTML/CSS/JS starter with no build tools.
- Perfect for a simple landing page, portfolio, or documentation homepage.
